1. Strukturierende Tags

  • html: Der Wurzel-Tag eines HTML-Dokuments.
  • head: Enthält Metadaten, wie title, meta, link und style.
  • title: Setzt den Titel der Webseite, der im Browser-Tab angezeigt wird.
  • body: Enthält den Hauptinhalt der Webseite.

2. Textformatierung

  • h1 bis h6: Überschriften-Tags. h1 ist die wichtigste (größte) Überschrift, h6 die unwichtigste (kleinste).
  • p: Absatz-Tag. Definiert einen Textabsatz.
  • br: Zeilenumbruch. Fügt einen einfachen Zeilenumbruch ein.
  • hr: Horizontale Linie. Fügt eine horizontale Linie ein, die als thematischer Trenner dient.
  • strong: Markiert wichtigen Text (normalerweise fett gedruckt).
  • em: Markiert betonten Text (normalerweise kursiv).
  • b: Fettgedruckter Text.
  • i: Kursiver Text.

3. Links und Bilder

  • a href="URL": Hyperlink. Verlinkt auf eine andere Seite oder einen anderen Bereich der aktuellen Seite.
  • img src="URL" alt="Beschreibung": Bild. Fügt ein Bild ein. src definiert die Bildquelle und alt den alternativen Text.

4. Listen

  • ul: Ungeordnete Liste (Bullet-Point-Liste).
  • ol: Geordnete Liste (nummerierte Liste).
  • li: Listenelement. Wird innerhalb von ul oder ol verwendet.
